Корпоративный чат

Приложение, написанное на фреймворке Windows Communication Foundation с использованием базы данных MySQL.

Проект был разработан для учебной дисциплины "Управление программными продуктами".

Проект - оконное приложение, предлагающее возможности управления рабочими чатами-каналами, обмена сообщениями в этих чатах.
